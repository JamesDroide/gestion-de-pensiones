# Design System — Gestión de Pensionistas

## Paleta de colores
- **Primary:** Azul pizarra (#6270f5) — acciones principales, sidebar activo
- **Surface:** Grises neutros (#f9fafb base) — fondos de página y tarjetas
- **Success:** Verde (#16a34a) — pagado, al día
- **Warning:** Ámbar (#d97706) — deuda pendiente, advertencias
- **Danger:** Rojo (#dc2626) — deuda alta, errores
- **Info:** Azul claro (#0284c7) — información neutral

## Tipografía
- **Familia:** Inter (sans-serif) — legible en pantallas de mostrador
- **Escala:** xs(12) / sm(14) / base(16) / lg(18) / xl(20) / 2xl(24) / 3xl(30)
- **Peso:** 400 texto normal | 500 etiquetas | 600 títulos de sección | 700 encabezados

## Espaciado
Sistema de 4px base: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64px

## Componentes clave
- **DataTable:** filas con hover sutil, sin bordes agresivos, paginación simple
- **StatusBadge:** pill redondeado con color semántico (al día / pendiente / deuda alta)
- **FormCard:** contenedor blanco con sombra xs, padding 24px
- **Sidebar:** fondo primary-950, íconos y texto en surface-50, activo con primary-500
- **TopBar:** blanco con sombra xs, nombre del sistema y datos del usuario logueado

## Principios de diseño
1. **Claridad sobre creatividad** — el cajero usa esto en servicio activo
2. **Jerarquía visual clara** — lo importante siempre visible sin hacer scroll
3. **Separación visual civil/policía** — badges de color distintos por tipo
4. **Densidad media** — suficiente información sin abrumar
5. **Responsive tablet-first** — pensado para uso en mostrador con tablet
